# Swiggy Look alike Application

#
- Install the npm packages required using npm i
- npm start / npm run start to view the application
- Make sure you have CORS installed in your browser to make API call to swiggy
- Shimmer UI is added to the intial cards to make better UX
- Routing with latest react-router-dom
